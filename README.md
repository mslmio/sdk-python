# Mslm Python SDK

The official Python SDK for the Mslm APIs.

## Installation

```bash
pip install mslm
```

## Usage

```python
from mslm import Mslm

m = Mslm("api_key")

resp = m.email_verify.single_verify("support@mslm.io")

print(resp)
```
